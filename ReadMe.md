# Emera Component Manager

This is a central repository for all things ECM. I may eventually use this as a registry, but for now it's for sharing the source code for the "official" ECM components as well as ECM itself 💚

## Roadmap

- [ ] Create an easy ECM Installer plugin
	- [ ] Add the plugin to Obsidian Community Plugins

## Contributing

The source code for everything ECM is in this vault! Feel free to clone and make pull requests! But maybe open up a ticket before you invest too much time into a big feature, to make sure that it aligns with the roadmap :)
